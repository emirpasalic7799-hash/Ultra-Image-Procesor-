# Ultra-Image-Procesor-
Besplatna Web Aplikacija za Upscaling Poboljšanje Rezolucije Fotografija 
Ultra Image Processor (Open-Source AI Enhancer)
Ultra Image Processor je lagan, ali moćan alat za obradu i poboljšanje slika koji radi u potpunosti u browseru. Cilj projekta je pružiti korisnicima privatnost (offline obrada) i visok kvalitet bez potrebe za skupim pretplatama.

🚀 Mogućnosti (Features)
High-Quality Upscale: Povećanje rezolucije pomoću napredne bicubic interpolacije.

Pro Sharpen & Deblur: Korištenje konvolucionih matrica (unsharp mask) za vraćanje oštrine.

Median Denoise: Efikasno uklanjanje digitalnog šuma uz očuvanje ivica.

100% Privacy: Sva obrada se vrši lokalno na vašem uređaju. Vaše slike nikada ne napuštaju vaš računar.

Multi-format Export: Podrška za PNG, JPG i WebP formate.

🛠️ Tehnologije (Tech Stack)
HTML5 & CSS3: Moderni "Glassmorphism" UI.

Vanilla JavaScript: Obrada piksela direktno preko Canvas API-ja bez eksternih biblioteka.

🎯 Planovi za budućnost (Roadmap)
Želim da ovaj projekat preraste u ozbiljnu desktop aplikaciju. Planovi su:

AI Upscaling: Integracija TensorFlow.js za pravi AI upscaling (ESRGAN/SwinIR).

Desktop Verzija: Pakovanje aplikacije u Electron.js (Windows, Mac, Linux).

Web Workers: Prebacivanje obrade u pozadinske procese kako bi UI ostao fluidan kod velikih slika.
